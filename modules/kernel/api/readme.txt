SpawnEngine Kernel - API Layer

Här hanteras alla API-anrop för SpawnEngine-plattformen:

meshAPI.js      → Hämtar/Pushar onchain events (One Event Mesh)
walletAPI.js    → Wallet-anslutningar, transaktioner och balans
creatorAPI.js   → Creators, packs och statistik
systemAPI.js    → Ping, status och systemloggar

Byt enkelt ut mock-data mot riktiga endpoints senare.
