SpawnEngine Kernel - Core Layer

Detta är hjärnan bakom SpawnEngine.

MeshKernel.jsx     → React-entry som startar motorn
MeshOrchestrator.js → Loopar Mesh-events, AI-insikter, och pulses
KernelInit.js      → Boot-sekvens (mockad systeminitiering)
SystemPulse.js     → UI-komponent för live mesh-pulser

Körs tillsammans med API-lagret och Context.
